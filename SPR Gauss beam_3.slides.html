Plane wave versus Real beam¶

(Ermolaev Georgy and Beniya Ilya)

Problem statement:¶

  • The problem of the plane wave interaction with the optical system is a linear problem (Transfer Matrix Method)
  • However the plane wave does not exist!!! In real situation beams have the other shape (ex. gaussian beam)

Solution:¶

The beam of the arbitrary shape could be presented in the angular representation (Fourier) $ \Rightarrow$ linear problem

Verification:¶

Outcomes:¶

  • Transfer Matrix Method was expanded from the plane wave to the real beam
  • The shape of the beam influences the response (R, T, A) of the optical system, especially in case of thin layers
  • This result is relevant for the modern nano optoelectronics
